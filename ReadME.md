<p align="center""><em>Mobilná aplikácia pre sledovanie spotrebných produktov v domácnosti</em></p>

> [!IMPORTANT]
> Toto je moja prvá semestrálna práca pre [**Univerzitu sv. Cyrila a Metóda v Trnave**](https://www.ucm.sk/) pre predmet _Úvod do štúdia informatiky_ (študijný odbor Aplikovaná informatika [Ústavu počítačových technológií a informatiky](https://upti.fpv.ucm.sk/) [**Fakulty prirodných vied**](https://fpv.ucm.sk/)).

> [!CAUTION]
> Na ReadME sa ešte pracuje, nemusia byť dokončené alebo v slovenčine.

## Funkcionality
* Pridávanie spotrebné produkty do zoznamu & databázy
* Sledovanie zostávajúcej spotreby produktov
* Dopĺňanie informácií z verejného API [OpenFoodFacts](https://openfoodfacts.org/)


## Build & Run (Expo Go)
Depedencies:
* [Node.js](https://nodejs.org/)
1. Start dev server:
    ```
    $ npx expo start
    ```
2. Open the app via *Expo Go* by scanning the QR code.

More info on [how to set up your environment (docs.expo.dev)](https://docs.expo.dev/get-started/set-up-your-environment/).
